# Bluetooth Headset Keeper

Bluetooth Headset Keeper 是一个简单的 Python 应用，它可以在你的电脑上播放一种几乎听不见的噪音，以防止蓝牙耳机因为没有声音输入而自动断开。

## 使用

运行程序后，你会在系统托盘中看到一个图标。右键点击图标，你会看到一个菜单：

- "Open"：打开一个小窗口，你可以在这个窗口中控制噪音的播放和停止。
- "Exit"：退出程序。

在 "Open" 打开的窗口中，你会看到三个按钮：

- "Play"：开始播放噪音。
- "Stop"：停止播放噪音。
- "Exit"：退出程序。
